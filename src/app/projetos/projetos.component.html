Api Rest - Cadastro de produtos

Descrição: A finalidade desse projeto é cadastar uma lista de produtos de um pequeno comércio.
Detalhes técnicos: Api Rest desenvolvida em Java com spring. Para fazer o crud criei métodos com Native query que podem ser vistos na interface repository, o projeto tem uma classe de configuração para instanciar alguns objetos para testes no banco H2 assim que rodar o projeto.
O projeto no momento só tem uma entidade (product), e segue o pradão MVC.

Repositório do projeto: https://github.com/alfdaniel/Cadastro-de-produtos
