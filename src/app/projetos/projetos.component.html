Cadastro de produtos

Descrição: A finalidade desse projeto é cadastar uma lista de produtos de um pequeno comércio,
Detalhes técnicos: Api Rest desenvolvida em Java com spring jpa e native query, o banco utilizado foi o h2, temos uma classe de configuração para instanciar alguns objetos para testes.
Repositório do projeto: https://github.com/alfdaniel/Cadastro-de-produtos
